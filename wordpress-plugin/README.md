# WordPress Plugins (HAMADO-CARD)

هذا المجلد يحتوي حالياً على بلوجنين:

1. `hamado-card-api.php` (واجهة REST الأساسية للمستخدمين والطلبات والمحفظة)
2. `sho8rday-wp.php` (أتمتة نشر قانونية مع OpenRouter + Telegram + SEO + سجلات Debug)

---

## sho8rday - WP

بلوجن أتمتة قانوني لمواقع المحتوى، يركز على النشر الاحترافي دون استخدام أي أساليب مخالفة (مثل السبام أو المراجعات المزيفة أو المحتوى المقرصن).

### المزايا
- حفظ الإعدادات مرة واحدة داخل WordPress options.
- دعم حتى **10 مفاتيح OpenRouter** (صيغة: `type|label|api_key`).
- تدوير تلقائي للنماذج والمفاتيح حتى العثور على نموذج يعمل.
- قائمة نماذج قابلة للتخصيص + نموذج مفضل.
- إنشاء صفحات تطبيقات عبر **Custom Post Type: `sho8_app`**.
- نشر تلقائي على تيليجرام (قناة/مجموعة عبر bot token).
- دعم Ad slots أعلى/أسفل المحتوى.
- دعم تحسين SEO الوصفي (meta description) للمنشورات الجديدة.
- سجل Debug مباشر داخل صفحة الإعدادات.
- Cron دوري (`hourly`) لتشغيل المهام التلقائية من الطابور.

### التثبيت
1. انسخ `sho8rday-wp.php` إلى:
   `wp-content/plugins/sho8rday-wp/sho8rday-wp.php`
2. فعّل الإضافة من لوحة التحكم.
3. افتح صفحة `sho8rday - WP` من القائمة الجانبية.

### REST Endpoints
- إضافة مهمة للطابور:
  - `POST /wp-json/sho8rday/v1/queue`
  - body: `topic=اسم التطبيق`
- تشغيل دورة الأتمتة يدويًا:
  - `POST /wp-json/sho8rday/v1/run`

> تتطلب صلاحية `manage_options`.

### ملاحظة قانونية
الإضافة مصممة للاستخدام القانوني فقط:
- لا تدعم النشر المقرصن/المهكر.
- لا تدعم السبام البريدي.
- لا تدعم مراجعات Google المزيفة أو أي تلاعب بالمنصات.
